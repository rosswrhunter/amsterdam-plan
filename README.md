# Amsterdam Marathon 2026 — Sub-4:00 Training Plan

A personal day-by-day marathon training plan built with React + Vite.

## Features
- 📅 Every day from 30 March → 18 October 2026
- 🏃 Phase-based training: Base → Build → Peak → Taper
- 🥗 Tap any day for full nutrition & macro breakdown
- 📊 Calibrated to real Garmin data (VO₂ max, lactate threshold)
- 🎯 Goal: Sub-4:00 at Amsterdam Marathon 2026

## Dev
```bash
npm install
npm run dev
```
